## متن فارسی نهایی

پیدا کردن یک API «واقعاً رایگان» برای مدل‌های زبانی فقط پیدا کردن یک فهرست نام نیست.

باید بدانیم سهمیه رایگان دائمی است یا Trial، کارت بانکی لازم دارد یا نه، با OpenAI SDK سازگار است یا نه، محدودیت RPM/RPD/TPM چقدر است و دسترسی منطقه‌ای آن چگونه ثبت شده است.

برای حل این مسئله، پروژه متن‌باز **Awesome Free LLM APIs IR** را آماده کرده‌ایم: یک فهرست فارسی و ماشین‌خوان از APIهای رایگان LLM با شواهد تاریخ‌دار، صفحات مستقل Provider، Guideهای کاربردی و Catalog JSON.

Repository:

اطلاعات این سرویس‌ها دائماً تغییر می‌کند. اگر Provider جدیدی می‌شناسید یا داده‌ای قدیمی شده، Issue یا Pull Request شما به کامل‌ترشدن پروژه کمک می‌کند.

## English final copy

Finding a “free” LLM API is not just about collecting provider names.

Developers also need to know whether the offer is a permanent free tier or a trial, whether a payment method is required, whether the endpoint is OpenAI-compatible, what the documented quotas are, and how regional-access evidence is classified.

We built **Awesome Free LLM APIs IR**, an open, Persian-first and machine-readable catalog of free LLM APIs with dated evidence, dedicated provider pages, practical guides and a JSON catalog.

Website:
https://llm.persiantoolbox.ir/?utm_source=linkedin&utm_medium=social&utm_campaign=international_launch

Repository:

Free-tier policies change frequently, so reports and contributions are welcome.

## Hermes UI steps

1. Account display name را ثبت و به مالک نشان بده.
2. `Start a post` را باز کن.
3. متن همان زبان را Paste کن.
4. Asset را Upload یا Link preview را تأیید کن.
5. بررسی کن Host لینک `llm.persiantoolbox.ir` است.
6. Preview نهایی شامل متن، تصویر، زبان و Account را نمایش بده.
7. قبل از `Post` متوقف شو.
8. Approval مستقل `APPROVE LINKEDIN FA` یا `APPROVE LINKEDIN EN` بگیر.
9. فقط همان Post را منتشر کن.
10. صفحه عمومی Post را باز و URL آن را استخراج کن.
11. ردیف Launch Log متناظر را به `PUBLISHED` تغییر بده.
