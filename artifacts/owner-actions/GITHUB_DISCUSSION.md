## Hermes UI steps

1. بررسی کن Discussions برای Repository فعال است.
2. Category مناسب مانند `Announcements` یا نزدیک‌ترین Category موجود را انتخاب کن.
3. Title و Body را Preview کن.
4. UTM و لینک Repository را کنترل کن.
5. قبل از `Start discussion` متوقف شو.
6. Approval `APPROVE GITHUB DISCUSSION` بگیر.
7. Discussion را منتشر کن.
8. Public URL را ثبت و ردیف `L-002` را `PUBLISHED` کن.
9. اگر Discussions غیرفعال است، خودسرانه Repository setting را تغییر نده؛ وضعیت `BLOCKED_DISCUSSIONS_DISABLED` ثبت کن.
