## Hermes UI steps

1. Account را به مالک نشان بده.
2. یکی از حالت‌های Single post یا Thread را انتخاب و اعلام کن.
3. Composer واقعی را برای محدودیت جاری متن ملاک قرار بده؛ متن را کورکورانه Trim نکن.
4. در Thread، ترتیب همه Postها را پیش از اولین انتشار Preview کن.
5. قبل از اولین `Post` متوقف شو.
6. Approval مستقل `APPROVE X FA`, `APPROVE X EN` یا `APPROVE X THREAD FA` بگیر.
7. پس از انتشار، Public URL اولین Post را ثبت کن.
8. Replyهای Thread را فقط طبق Preview تأییدشده منتشر کن.
9. هیچ Tag، Mention یا درخواست Repost/Like اضافه نکن.
