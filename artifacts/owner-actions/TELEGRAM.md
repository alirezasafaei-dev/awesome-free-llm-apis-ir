## Hermes UI steps

1. نام Channel/Group مقصد را به مالک نشان بده.
2. تأیید کن مقصد، کانال رسمی یا Community مجاز است.
3. تصویر و متن را به‌صورت Draft آماده کن.
4. Link preview و UTM را کنترل کن.
5. قبل از `Send` متوقف شو.
6. Approval دقیق `APPROVE TELEGRAM <DESTINATION>` بگیر.
7. فقط یک مقصد را ارسال کن.
8. Public message URL را از `Copy Message Link` استخراج کن.
9. اگر Channel خصوصی یا بدون Public URL است، آن را Publication عمومی حساب نکن و `BLOCKED_NO_PUBLIC_URL` ثبت کن.

## Pre-publish checklist

- [ ] مقصد دقیق تأیید شده
