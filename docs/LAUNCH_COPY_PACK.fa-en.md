# ۱۷. متن خبرنامه یا رسانه

## Subject

Open-source catalog of free LLM APIs with regional-access evidence

## Pitch

Awesome Free LLM APIs IR is an open and machine-readable catalog that separates permanent free tiers, trials, credits and free models. It also tracks documented quotas, payment requirements, OpenAI compatibility, Base URLs and dated access evidence for users in restricted regions.

The same validated dataset powers the website, provider pages and JSON catalog. The project may be useful to developers comparing low-cost inference options or researchers studying regional API availability.

Website: https://llm.persiantoolbox.ir/
GitHub: https://github.com/alirezasafaei-dev/awesome-free-llm-apis-ir

Press/social assets are available under `assets/social/` in the Repository.

# ۱۸. قواعد انتشار

- هیچ Password، Token، IP، Username زیرساخت، Recovery Code یا URL مدیریتی منتشر نشود.
- ادعای «بهترین» یا «کاملاً رایگان» بدون معیار روشن استفاده نشود.
- تعداد Providerها پیش از هر انتشار با Catalog زنده تطبیق داده شود.
- برای هر کانال UTM مخصوص همان کانال استفاده شود.
- پیام‌ها بر اساس قوانین و مخاطب هر Community شخصی‌سازی شوند.
- درخواست Upvote هماهنگ‌شده، خرید Backlink، Comment spam و ارسال انبوه ممنوع است.
- لینک و نتیجه هر انتشار در `docs/LAUNCH_LOG.md` ثبت شود.
