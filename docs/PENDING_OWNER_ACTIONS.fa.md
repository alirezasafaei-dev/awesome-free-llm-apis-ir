# اقدامات باقی‌ماندهٔ مالک

آخرین بازبینی: ۲۰۲۶-۰۷-۱۶

این موارد به حساب یا دسترسی مالک نیاز دارند و نباید با Automation دور زده شوند:

- ساخت حساب مالک Analytics با Credential جدید و یکتا
- افزودن دامنه Canonical به Dashboard
- بستن Registration عمومی پس از Bootstrap
- فعال‌سازی 2FA و نگهداری Recovery code
- بررسی Pageview و Custom Event در Dashboard
- Google Search Console و ارسال Sitemap
- Bing Webmaster Tools

Dashboard Analytics نباید از URL خام HTTP روی IP عمومی Bootstrap شود. استفاده از HTTPS معتبر یا SSH tunnel الزامی است.
