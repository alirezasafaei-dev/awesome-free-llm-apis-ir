# گیت نهایی Analytics

Analytics فقط زمانی `verified` است که:

- Revision Production با `main` برابر باشد؛
- Tracker در Home، Provider و Guide بارگذاری شود؛
- Event Drop نشود؛
- Pageview در Dashboard دیده شود؛
- Custom Event در Dashboard دیده شود؛
- Registration عمومی بسته باشد؛
- Credential افشاشده Rotate شده باشد.
