# خلاصه وضعیت دسترسی ایران

وضعیت Canonical شامل ۹ سرویس `verified_working`، پنج سرویس `verified_blocked`، دو سرویس `signup_blocked`، یک سرویس `officially_unsupported` و پنج سرویس `unknown` است.

پاسخ یک Endpoint فقط Reachability را نشان می‌دهد و به‌تنهایی موفقیت مدل یا اعتبار حساب را اثبات نمی‌کند. خطای اتصال یا Timeout نیز به‌تنهایی اثبات محدودیت منطقه‌ای نیست.

Fireworks AI تا زمان وجود تست معتبر با حساب فعال در وضعیت `unknown` باقی می‌ماند.
