# اجرای تست واقعی از داخل ایران

این ابزار دو مسیر را جداگانه ثبت می‌کند: اتصال مستقیم ایران و اتصال از ایران با VPN. تست VPN معتبر است، اما هرگز جای تست مستقیم را نمی‌گیرد و باید با کشور خروجی جدا ثبت شود. GitHub-hosted runner نتیجهٔ محلی ایران تولید نمی‌کند.

## آماده‌سازی

```bash
cp .env.example .env
```

در `.env` فقط کلید سرویس‌هایی را قرار دهید که مجاز به استفاده از آن‌ها هستید. فایل `.env` در Git نادیده گرفته می‌شود و نباید ارسال یا Screenshot شود.

اطلاعات شبکه را بدون IP کامل وارد کنید:

```dotenv
IR_TEST_COUNTRY=IR
IR_TEST_ROUTE=direct
IR_TEST_EXIT_COUNTRY=IR
IR_TEST_VPN_PROVIDER=
IR_TEST_ISP=نام اپراتور
IR_TEST_ASN=AS12345
IR_TEST_CITY=
```

ابتدا تنظیمات را بررسی کنید:

```bash
npm run verify:iran:dry
```

سپس یک یا چند سرویس را با مصرف حداکثر ۸ توکن تست کنید:

```bash
npm run verify:iran -- --providers=openrouter,groq
```

برای تست دوم با VPN، پس از ذخیرهٔ گزارش مستقیم، VPN مجاز خود را فعال و مسیر را دقیق ثبت کنید:

```dotenv
IR_TEST_ROUTE=vpn
IR_TEST_EXIT_COUNTRY=DE
IR_TEST_VPN_PROVIDER=نام عمومی سرویس
```

کشور خروجی باید مقدار مشاهده‌شده باشد. موفقیت VPN نباید وضعیت سیاست رسمی یا دسترسی مستقیم را بازنویسی کند.

خروجی پاک‌سازی‌شده در `reports/local/` ذخیره می‌شود. این پوشه عمداً Gitignore است. گزارش خام را Commit نکنید؛ ابتدا نتیجه را بررسی و فقط خلاصهٔ لازم را در Issue یا PR وارد کنید.

## کنترل اعتبار کلید

برای ادعای «مسدود با IP ایران»، همان کلید باید پیش‌تر از یک منطقهٔ پشتیبانی‌شده معتبر شناخته شده باشد. در این صورت متغیر `<PROVIDER>_CREDENTIAL_VALIDATED=true` تنظیم می‌شود. خطای 401، 403 عمومی، Timeout، 429 یا خرابی DNS به‌تنهایی مدرک Geo-block نیست.

## حریم خصوصی

اسکریپت IP، کلید، Headerهای پاسخ، متن پاسخ یا پیام کامل خطا را ذخیره نمی‌کند. فقط کد HTTP، دستهٔ نتیجه، زمان پاسخ و Hash بدنه ثبت می‌شوند.

### شبکه

- **تست مستقیم (۲۰۲۶-۰۷-۱۶)**: اینترنت موبایل ایران (MCI, AS44244)
- **ASN مستقل**: در دسترس نیست (فقط AS44244 موجود است)
- **VPN**: در دسترس نیست — ماتریس VPN اجرا نشده
- **اعتبارسنجی کلید**: نیاز به تنظیم API Key در فایل `.env`

### محدودیت‌های فعلی

1. **عدم دسترسی به API Key**: ۱۰ سرویس نیاز به کلید API دارند
2. **عدم دسترسی به ASN مستقل**: فقط AS44244 (MCI) در دسترس است
3. **عدم دسترسی به VPN**: ماتریس VPN اجرا نشده

### اقدامات بعدی

برای تکمیل تست سرویس‌های باقی‌مانده:

```bash
# ۱. کلیدهای API را در فایل .env تنظیم کنید
# ۲. تست مستقیم را اجرا کنید
npm run verify:iran -- --providers=groq,nvidia-nim,siliconflow,modelscope,opencode-zen,agnes-ai,aion-labs,freetheai,fireworks-ai,vercel-ai-gateway

# ۳. VPN را فعال کنید و مسیر را تغییر دهید
# IR_TEST_ROUTE=vpn
# IR_TEST_EXIT_COUNTRY=<کشور خروجی>
# IR_TEST_VPN_PROVIDER=<نام سرویس>

# ۴. تست VPN را اجرا کنید
npm run verify:iran
```
