# Launch Log

این فایل مرجع ثبت انتشارها و نتایج کمپین‌های پروژه **Awesome Free LLM APIs IR** است.

## قواعد

- هر ردیف فقط یک انتشار عمومی را ثبت کند.
- `Public URL` باید لینک واقعی همان پست، مقاله، ویدئو یا Discussion باشد.
- UTM باید با کانال تطابق داشته باشد.
- هیچ Credential، Token، IP، Username زیرساخت یا داده شخصی در این فایل ثبت نشود.
- آمار در بازه‌های `24h`، `72h` و `7d` ثبت شوند.
- آمار ناموجود با `N/A` مشخص شود؛ مقدار حدسی وارد نشود.

## وضعیت‌های مجاز

- `DRAFT_READY`
- `REVIEWED`
- `SCHEDULED`
- `PUBLISHED`
- `MONITORING`
- `COMPLETED`
- `BLOCKED`
