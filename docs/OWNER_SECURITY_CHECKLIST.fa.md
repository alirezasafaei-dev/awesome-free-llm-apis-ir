# Checklist امنیتی مالک

- [ ] Credential افشاشده استفاده نشده یا Rotate شده است
- [ ] Dashboard فقط از HTTPS یا SSH tunnel قابل دسترسی است
- [ ] Registration عمومی بسته است
- [ ] 2FA فعال است
- [ ] Sessionهای فعال Review شده‌اند
- [ ] Port داخلی Dashboard در Firewall عمومی باز نیست
- [ ] هیچ Secret در GitHub یا Log عمومی وجود ندارد
