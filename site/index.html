API رایگان هوش مصنوعی
بدون حدس و تبلیغ.

API راهی است که برنامه، سایت یا ربات شما را به یک مدل هوش مصنوعی وصل می‌کند. سرویس‌ها را بر اساس سهمیه، مدل، فرمت OpenAI، شرایط حساب و شواهد دسترسی ایران بررسی و مقایسه کن.

برای چت‌بات، تولید متن، کدنویسی، استدلال، Embedding و ابزارهای Agent

برای من API انتخاب کن اولین درخواست را بساز همه سرویس‌ها
Evidence-firstنتیجه شبکه، ثبت‌نام، کلید و استنتاج از هم جدا ثبت می‌شوند.
محدودیت واقعیRPM، RPD، اعتبار ماهانه و نیاز به روش پرداخت قابل مشاهده است.
منبع و تاریخهر ادعا باید به منبع رسمی یا آزمایش محدودشده متصل باشد.

    تعریف ساده

    API هوش مصنوعی چیست؟

    برنامه تو یک درخواست شامل متن و تنظیمات برای سرویس هوش مصنوعی می‌فرستد و پاسخ مدل را دریافت می‌کند. به‌جای ساخت مدل از صفر، از توان مدل آماده داخل سایت، ربات یا ابزار خودت استفاده می‌کنی.

    قبل از انتخاب

    چهار موضوع را با هم قاطی نکن

    بازشدن سایت، ساخت حساب، دریافت کلید و پاسخ‌گرفتن از مدل چهار مرحله جدا هستند.

    رایگان یک معنی ندارد

    سهمیه دائمی، مدل رایگان، اعتبار ماهانه و Trial شرایط متفاوتی دارند.

    دسترسی صفحه کافی نیست

    ممکن است سایت باز شود اما ثبت‌نام، کلید یا استنتاج روی همان حساب محدود باشد.

    تاریخ Evidence مهم است

    قیمت، محدودیت و سیاست منطقه‌ای تغییر می‌کند؛ تاریخ آخرین بررسی را بخوان.

    واژه‌نامه

    اصطلاحات مهم کارت‌ها

    API

    راه ارتباط برنامه با سرویس مدل هوش مصنوعی.

    LLM

    مدل زبانی بزرگ برای فهم و تولید متن.

    Provider

    شرکت یا سرویسی که دسترسی API را ارائه می‌دهد.

    سازگار با OpenAI

    ساختار درخواست شبیه API رایج OpenAI است و معمولاً با تغییر Base URL و کلید قابل استفاده است.

    RPM و RPD

    تعداد درخواست مجاز در دقیقه و روز؛ این اعداد ظرفیت هستند، نه سرعت پاسخ.

    وضعیت ایران

    شبکه، ثبت‌نام، فعال‌سازی حساب و استنتاج جداگانه گزارش می‌شوند.

    برای استفاده از این سایت باید برنامه‌نویس باشم؟

    برای مقایسه گزینه‌ها نه؛ استفاده از API داخل برنامه به دانش پایه برنامه‌نویسی نیاز دارد.

    API رایگان دائمی با Trial چه تفاوتی دارد؟

    Trial پس از زمان یا اعتبار مشخص تمام می‌شود؛ سهمیه دائمی تا زمان تغییر سیاست سرویس قابل استفاده است.

    آیا همه APIها از ایران در دسترس‌اند؟

    خیر. نتیجه شبکه، ثبت‌نام، فعال‌سازی حساب و اجرای مدل برای هر سرویس جدا بررسی می‌شود.

    سازگار با OpenAI یعنی چه؟

    ساختار درخواست مشابه است و معمولاً با تغییر Base URL و کلید می‌توان از SDK مشابه استفاده کرد.

    داده‌ای اشتباه یا قدیمی است؟

    با یک گزارش دقیق کمک کن.

    منبع رسمی، تاریخ و نوع آزمایش را ثبت کن؛ بدون IP کامل، کلید API یا اطلاعات حساب.
