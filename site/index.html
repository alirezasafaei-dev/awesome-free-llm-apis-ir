    دسترسی به صفحه، ساخت حساب، گرفتن کلید و دریافت پاسخ مدل باید جداگانه بررسی شوند.

    منبع و تاریخ را کنترل کن

    سیاست‌ها و محدودیت‌ها تغییر می‌کنند؛ صفحه هر سرویس منبع رسمی و تاریخ آخرین بررسی را نشان می‌دهد.

    واژه‌نامه کوتاه

    اصطلاحات روی کارت‌ها یعنی چه؟

    API

    راه ارتباط برنامه شما با یک سرویس دیگر؛ در این سایت، راه ارتباط با مدل هوش مصنوعی.

    LLM

    مدل زبانی بزرگ؛ مدلی که متن را می‌فهمد یا تولید می‌کند، مانند مدل‌های چت و کدنویسی.

    Provider

    شرکت یا سرویسی که دسترسی API را ارائه می‌دهد.

    سازگار با OpenAI

    ساختار درخواست شبیه API رایج OpenAI است و معمولاً با تغییر آدرس و کلید می‌توان از SDK مشابه استفاده کرد.

    RPM و RPD

    تعداد درخواست مجاز در دقیقه و روز. این اعداد ممکن است برای هر مدل یا حساب متفاوت باشند.

    وضعیت ایران

    نتیجه شبکه، ثبت‌نام و اجرای مدل از هم جدا نگه داشته می‌شوند؛ برچسب هر سرویس را با تاریخ و شواهد بخوان.

    پرسش‌های پرتکرار

    سؤالات رایج درباره API رایگان هوش مصنوعی

    برای استفاده از این سایت باید برنامه‌نویس باشم؟

    برای فهمیدن گزینه‌ها نه؛ اما استفاده از API داخل یک برنامه معمولاً به دانش پایه برنامه‌نویسی نیاز دارد. مسیر تازه‌کار مفاهیم را ساده توضیح می‌دهد.

    API رایگان دائمی با Trial چه تفاوتی دارد؟

    Trial پس از زمان یا اعتبار مشخص تمام می‌شود؛ سهمیه دائمی یا مدل رایگان تا زمانی که سیاست سرویس تغییر نکند قابل استفاده است.

    آیا همه APIهای رایگان از ایران در دسترس‌اند؟

    خیر. وضعیت شبکه، ثبت‌نام، دریافت کلید و اجرای مدل جداگانه بررسی می‌شوند و نتیجه هر سرویس در صفحه اختصاصی آن درج شده است.

    سازگار با OpenAI یعنی چه؟

    یعنی ساختار درخواست سرویس با API رایج OpenAI سازگار است و معمولاً با تغییر Base URL و کلید می‌توان از SDKهای مشابه استفاده کرد.

    داده‌ای اشتباه یا قدیمی است؟

    با یک گزارش دقیق کمک کن.

    منبع رسمی، تاریخ و نوع آزمایش را ثبت کن؛ بدون IP کامل، کلید API یا اطلاعات حساب.
