How to choose a free LLM API

  1. 1Define your use case

    Chat, coding, reasoning, or embeddings — each capability maps to different providers and free tiers.

  2. 2Compare free tiers

    Permanent allowance, free models, monthly credits, or trial. Check if a payment method is required.

  3. 3Review evidence layers

    Reachability, signup, key issuance, and inference are four separate claims. Know what each badge means.

Separated evidence

Why you should trust this catalog

Every provider entry distinguishes between network reachability, signup completion, API key issuance, and authenticated inference. These four layers are never mixed. If only reachability is confirmed, the inference status remains unknown. No plausible guesses replace missing evidence.
