# بنچمارک پایهٔ فارسی

این پوشه یک آزمون کوچک، نسخه‌دار و قابل بازتولید برای سنجش حداقل توانایی مدل‌های رایگان در زبان فارسی است. نسخهٔ `v1` عمداً فقط از سؤال‌هایی استفاده می‌کند که بدون «داور مدل زبانی» و با قواعد قطعی قابل امتیازدهی‌اند؛ بنابراین نتیجه به سلیقهٔ داور یا سرویس پولی دیگری وابسته نیست.

## چه چیزهایی سنجیده می‌شوند؟

- پیروی دقیق از دستور فارسی
- درک متن کوتاه فارسی
- محاسبه و استدلال عددی پایه
- خروجی ساخت‌یافتهٔ JSON
- نویسه‌ها، فاصله‌گذاری و ارقام فارسی

این بنچمارک معیار جامع کیفیت نگارش، دانش عمومی یا ایمنی مدل نیست. نتیجه فقط باید با شناسهٔ نسخه، مدل دقیق، ارائه‌دهنده و زمان اجرا منتشر شود.

## اجرا

ابتدا چهار متغیر زیر را در `.env` قرار دهید. آدرس پایه باید به مسیر سازگار با OpenAI ختم شود؛ Runner مسیر `chat/completions` را به آن اضافه می‌کند.

```dotenv
BENCHMARK_PROVIDER_ID=groq
BENCHMARK_API_BASE_URL=https://api.groq.com/openai/v1
BENCHMARK_API_KEY=...
BENCHMARK_MODEL=llama-3.1-8b-instant
```

سپس:

```bash
npm run benchmark:validate
npm run benchmark:dry
npm run benchmark:run
```

برای اجرای چند سؤال یا تعیین محل خروجی:

```bash
npm run benchmark:run -- --prompts=instruction-01,reasoning-02 --output=benchmarks/results/local/my-run.json
```

خروجی خام در `benchmarks/results/local/` ذخیره و به‌صورت پیش‌فرض Commit نمی‌شود. هیچ کلید API یا Header احراز هویت در گزارش نوشته نمی‌شود. پیش از انتشار عمومی، پاسخ‌ها را از نظر دادهٔ شخصی بازبینی کنید.

## قواعد انتشار نتیجه

1. فایل‌های `manifest.json` و `prompts.json` همان نسخه نباید تغییر کرده باشند.
2. نام Provider و شناسهٔ دقیق مدل الزامی است.
3. Temperature باید صفر و تمام Promptها اجرا شده باشند.
4. شکست شبکه و Rate limit امتیاز صفر نیست؛ اجرای ناقص است و نباید در رتبه‌بندی نهایی قرار گیرد.
5. فقط Summary کافی نیست؛ گزارش JSON کامل و قابل ممیزی باید همراه نتیجه باشد.

محتوای این Dataset توسط مشارکت‌کنندگان همین پروژه نوشته شده و تحت مجوز MIT مخزن است.
