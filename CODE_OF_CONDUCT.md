# آیین رفتار

مشارکت‌کنندگان باید محترمانه، دقیق و متمرکز بر شواهد رفتار کنند. توهین، آزار، افشای اطلاعات شخصی، انتشار secret، تبلیغ پنهان و گزارش ساختگی پذیرفته نیست.

اختلاف فنی باید با مستندات و مراحل بازتولید بررسی شود. نگهدارندگان می‌توانند محتوای ناامن، گمراه‌کننده یا خارج از محدوده را حذف کنند.
