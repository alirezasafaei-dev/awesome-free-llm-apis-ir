# Awesome Free LLM APIs IR

A free, Persian-first, machine-readable catalog of free LLM APIs, focused on quotas, token limits, payment requirements, and access from Iranian IP addresses.

> [!IMPORTANT]
> A free tier does not imply availability in Iran. Iran access is recorded only when backed by official policy or dated real-world evidence. `Unknown` means there is not enough evidence; it does not mean the service works.

The canonical catalog is in [`data/providers`](data/providers), validated against [`schema/provider.schema.json`](schema/provider.schema.json). The main [Persian README](README.md) contains the generated summary table and contribution guidance.

## Principles

- Official documentation for quota claims
- Dated evidence for Iran access claims
- No API keys or personal data in issues or pull requests
- Machine-readable, website-ready data
- Stale-data warnings instead of silent assumptions
