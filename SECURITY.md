# Security Policy

این مخزن هیچ کلید API یا Secret عملیاتی نگهداری نمی‌کند، اما کد سایت ایستا، Analytics سمت Client و ابزارهای انتشار VPS را شامل می‌شود. اگر کلید، توکن، IP کامل، نام کاربری زیرساخت یا اطلاعات شخصی را در Issue یا Pull Request دیدید، آن را بازنشر نکنید و از طریق یک گزارش خصوصی امنیتی GitHub به نگهدارنده اطلاع دهید.

آسیب‌پذیری‌های سایت، Workflowهای GitHub Actions، اسکریپت‌های انتشار، تنظیمات وب‌سرور و نشت احتمالی داده‌های زیرساخت در محدودهٔ گزارش امنیتی این پروژه هستند. گزارش عمومی حاوی جزئیات قابل سوءاستفاده نسازید.

برای آسیب‌پذیری خود ارائه‌دهندگان، مستقیماً با تیم امنیتی همان سرویس تماس بگیرید. این مخزن مسئول یا نمایندهٔ ارائه‌دهندگان فهرست‌شده نیست.
