# Security Policy

این مخزن هیچ کلید API یا سرویس اجرایی نگهداری نمی‌کند. با این حال، اگر کلید، توکن، IP کامل یا اطلاعات شخصی را در Issue یا Pull Request دیدید، آن را بازنشر نکنید و از طریق یک گزارش خصوصی امنیتی GitHub به نگهدارنده اطلاع دهید.

برای آسیب‌پذیری خود ارائه‌دهندگان، مستقیماً با تیم امنیتی همان سرویس تماس بگیرید. این مخزن مسئول یا نمایندهٔ ارائه‌دهندگان فهرست‌شده نیست.
